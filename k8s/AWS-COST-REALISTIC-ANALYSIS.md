## 🏢 온프레미스 vs AWS 비용 구조

### 온프레미스 비용 구조

#### 1. 초기 투자 (One-time)
```
서버 하드웨어: $50,000 - $100,000
네트워크 장비: $10,000 - $20,000
스토리지: $20,000 - $40,000
랙/전력/냉각: $10,000 - $20,000
총 초기 투자: $90,000 - $180,000
```

#### 2. 월간 운영 비용
```
하드웨어 상각 (3년 기준): $2,500 - $5,000/월
전기/냉각: $500 - $1,000/월
인터넷 회선: $200 - $500/월
IDC/공간 비용: $500 - $1,500/월 (있는 경우)
하드웨어 유지보수: $200 - $500/월
소프트웨어 라이선스: $0 - $500/월
총 월간 비용: $3,900 - $9,000/월
```

#### 3. 인건비 (운영)
```
인프라 엔지니어: $5,000 - $10,000/월 (전문 인력)
또는
기존 팀원의 인프라 관리 시간: 20-40시간/월 x $50 = $1,000 - $2,000/월
```

**온프레미스 총 비용 (하드웨어 상각 포함)**: $4,900 - $11,000/월

### AWS 비용 구조

#### 1. 초기 투자
```
마이그레이션 비용: $5,000 - $10,000 (일회성)
교육/학습: $2,000 - $5,000 (일회성)
총 초기 투자: $7,000 - $15,000
```

#### 2. 월간 운영 비용
```
EKS 클러스터: $73/월
컴퓨팅 (노드): $300 - $500/월
데이터베이스 (DocumentDB, ElastiCache 등): $400 - $600/월
스토리지 (EBS, S3): $100 - $200/월
네트워크 (ALB, NAT Gateway, 데이터 전송): $100 - $200/월
관찰성 (CloudWatch, X-Ray): $100 - $200/월
기타 (ECR, Secrets Manager 등): $50 - $100/월
총 월간 비용: $1,123 - $1,873/월
```

#### 3. 인건비 (운영)
```
인프라 관리 시간: 5-10시간/월 x $50 = $250 - $500/월
(관리형 서비스로 인한 시간 절감)
```

**AWS 총 비용**: $1,373 - $2,373/월

## 🎯 AWS가 유리한 기준

### 1. 하드웨어 교체/업그레이드 시점

**시나리오**: 하드웨어가 3-5년 이상 사용되어 교체가 필요한 경우

```
온프레미스:
- 새 하드웨어 구매: $90,000 - $180,000
- 마이그레이션 비용: $10,000 - $20,000
- 총: $100,000 - $200,000 (일회성)

AWS:
- 마이그레이션 비용: $7,000 - $15,000 (일회성)
- 월간 비용: $1,373 - $2,373/월

3년 기준:
- 온프레미스: $100,000 (하드웨어) + $4,900 x 36 = $276,400
- AWS: $15,000 (마이그레이션) + $2,373 x 36 = $100,428

절감: $175,972 (64% 절감)
```

**✅ AWS가 유리**: 하드웨어 교체 시점에 AWS 전환 권장

---

### 2. 확장성 요구사항

**시나리오**: 트래픽이 급증하거나 불규칙한 경우

```
온프레미스:
- 피크 트래픽 대비 하드웨어 구매: $50,000 - $100,000
- 평상시 리소스 낭비: 50-70%
- 확장 시 새 하드웨어 구매 필요

AWS:
- Auto Scaling으로 자동 확장/축소
- 사용한 만큼만 비용 지불
- Spot Instance로 70% 비용 절감 가능

예시 (트래픽 2배 증가):
- 온프레미스: 새 하드웨어 $50,000 + 운영비 증가
- AWS: 월간 비용만 2배 증가 ($2,746 - $4,746/월)
```

**✅ AWS가 유리**: 확장성이 중요한 경우

---

### 3. 고가용성 요구사항

**시나리오**: 99.9% 이상 가동률이 필요한 경우

```
온프레미스:
- 이중화 하드웨어: $100,000 - $200,000
- 다중 데이터센터: $200,000 - $400,000
- DR 사이트: $50,000 - $100,000
- 총: $350,000 - $700,000

AWS:
- Multi-AZ 자동 구성: 포함
- 자동 백업 및 복구: 포함
- 추가 비용: $200 - $500/월 (Multi-AZ)
```

**✅ AWS가 유리**: 고가용성이 중요한 경우

---

### 4. 인건비가 높은 경우

**시나리오**: 전문 인프라 엔지니어가 필요한 경우

```
온프레미스:
- 인프라 엔지니어: $5,000 - $10,000/월
- 또는 기존 팀원의 20-40시간/월 = $1,000 - $2,000/월

AWS:
- 관리형 서비스로 인한 시간 절감: 5-10시간/월 = $250 - $500/월
- 절감: $750 - $1,750/월
```

**✅ AWS가 유리**: 인건비가 높은 경우

---

### 5. 보안/컴플라이언스 요구사항

**시나리오**: 높은 보안/컴플라이언스 요구사항

```
온프레미스:
- 보안 솔루션 구매: $20,000 - $50,000
- 보안 전문가: $3,000 - $6,000/월
- 감사/인증 비용: $10,000 - $30,000/년

AWS:
- GuardDuty, Security Hub: $100 - $300/월
- Config, CloudTrail: $50 - $200/월
- 컴플라이언스 인증: 포함 (SOC, ISO 등)
```

**✅ AWS가 유리**: 보안/컴플라이언스 요구사항이 높은 경우

---

### 6. 개발/테스트 환경

**시나리오**: 개발/테스트 환경이 자주 생성/삭제되는 경우

```
온프레미스:
- 개발 서버 구매: $20,000 - $40,000
- 평상시 사용률: 20-30% (낭비)

AWS:
- 필요할 때만 생성: $100 - $300/월
- 사용하지 않을 때 종료: $0
- Spot Instance: 70% 할인
```

**✅ AWS가 유리**: 개발/테스트 환경이 유동적인 경우

## 📊 상황별 비용 비교

### 시나리오 1: 하드웨어가 새로 구매된 경우 (1년 이내)

```
온프레미스:
- 하드웨어 상각: $2,500 - $5,000/월
- 운영 비용: $1,400 - $4,000/월
- 인건비: $1,000 - $2,000/월
총: $4,900 - $11,000/월

AWS:
- 클라우드 비용: $1,123 - $1,873/월
- 인건비: $250 - $500/월
총: $1,373 - $2,373/월

비용 차이: AWS가 $3,527 - $8,627/월 더 비쌈
```

**결론**: ❌ **온프레미스가 유리** - 하드웨어가 새로 구매된 경우 온프레미스 유지 권장

---

### 시나리오 2: 하드웨어 교체가 필요한 경우 (3년 이상)

```
온프레미스 (3년):
- 하드웨어 구매: $100,000 (일회성)
- 운영 비용: $4,900 x 36 = $176,400
- 인건비: $1,000 x 36 = $36,000
총: $312,400

AWS (3년):
- 마이그레이션: $15,000 (일회성)
- 클라우드 비용: $1,873 x 36 = $67,428
- 인건비: $500 x 36 = $18,000
총: $100,428

절감: $211,972 (68% 절감)
```

**결론**: ✅ **AWS가 유리** - 하드웨어 교체 시점에 AWS 전환 권장

---

### 시나리오 3: 확장성이 중요한 경우

```
온프레미스:
- 피크 대비 하드웨어: $50,000
- 평상시 낭비: 50% = $25,000
- 3년 운영: $176,400
총: $251,400

AWS:
- Auto Scaling: 사용한 만큼만
- Spot Instance 70% 활용
- 3년 운영: $67,428
총: $67,428

절감: $183,972 (73% 절감)
```

**결론**: ✅ **AWS가 유리** - 확장성이 중요한 경우

### 시나리오 4: 소규모 팀 (4명), 하드웨어 이미 보유

```
온프레미스:
- 하드웨어 상각: $2,500/월 (이미 구매됨)
- 운영 비용: $1,400/월
- 인건비: $1,000/월 (기존 팀원 시간)
총: $4,900/월

AWS:
- 클라우드 비용: $1,373/월
- 인건비: $250/월
총: $1,623/월

비용 차이: AWS가 $3,277/월 더 저렴
```

**하지만**: 하드웨어가 이미 구매되어 있고, 상각 기간이 남아있다면?

```
실제 추가 비용:
- 온프레미스: $1,400/월 (운영) + $1,000/월 (인건비) = $2,400/월
- AWS: $1,373/월 (클라우드) + $250/월 (인건비) = $1,623/월

절감: $777/월 (32% 절감)
```

**결론**: ⚠️ **조건부** - 하드웨어 상각 기간이 남아있으면 온프레미스 유지, 교체 시점에 AWS 전환

### 1. 하드웨어가 새로 구매된 경우
- 하드웨어 상각 기간이 남아있음

- 기존 팀원이 인프라 관리 가능
- 전문 인프라 엔지니어 불필요
